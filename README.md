# AI Chatbot with Learning Capabilities

## Overview
This project is an interactive chatbot that can respond to user questions and learn new information over time.

## Objectives
- Build a chatbot that interacts with users
- Allow the system to learn new answers
- Integrate AI for intelligent responses

## Technologies Used
- Python
- OpenAI API
- Speech-to-text (STT)
- Text-to-speech (TTS)
- JSON-based knowledge base

## Features
- Responds to user questions
- Learns new answers from users
- Uses AI when knowledge is missing
- Supports both text and voice interaction

## Skills Demonstrated
- Python programming
- API integration
- AI-based system design
- Data handling and storage
